# Rubicon Slide Deck

Self-contained HTML slide deck for Rubicon. No build step, no dependencies.

## View it

Open `slides/rubicon.html` directly in any browser, or serve the folder:

```bash
python -m http.server --directory slides 8080
# then open http://localhost:8080/rubicon.html
```

## Navigate

- `←` / `→` — previous / next slide
- `Space` / `PageDown` — next
- `Home` / `End` — first / last
- Deep link a specific slide with `#N` in the URL (e.g. `rubicon.html#7`)
